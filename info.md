# Simple Compact Appliances

A minimalist, themable Lovelace card for controlling washers, dryers, dishwashers, and microwaves from a single compact card.

- One Control tab per appliance, one Summary tab listing them all
- Per-control entity mapping (switch / sensor / select / number / button)
- Auto-discovery from a `device_id` with per-control overrides
- Full GUI editor — no YAML needed for the common case
- Material You-style palette routed through CSS variables for theming

See the [README](https://github.com/priyam13coding/simple-compact-appliances-card) for the full YAML reference and theming guide.
